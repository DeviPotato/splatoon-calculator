# splatoon-calculator

splatoon-calculator.herokuapp.com

App is in 
```
client/app
```

## Usage

Build with:
```
grunt -force
```

Load live Preview:
```
grunt serve
```
See https://github.com/DaftMonk/generator-angular-fullstack for more information.
